# Chatbot_FAQ
**⚠️ Disclaimer:** **For security reasons, the Gemini API key is not included in this repository. Please add your own API key in the `.env` file before running the project.**

An AI-powered FAQ Chatbot built using **Python**, **Flask**, **Google Gemini API**, **HTML**, **CSS**, and **JavaScript**. This chatbot provides instant and intelligent responses to frequently asked questions through a clean and user-friendly web interface.

---

## ✨ Features

- 💬 AI-powered chatbot using Google Gemini API
- 🌐 Simple and responsive web interface
- ⚡ Fast and accurate responses
- 🖥️ Flask-based backend
- 🎨 Clean UI with HTML, CSS, and JavaScript
- 🔒 Secure API key management using `.env`

## 🚀 Installation

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/Chatbot_FAQ.git
cd Chatbot_FAQ
```

### 2. Install Dependencies

```bash
pip install -r requirements.txt
```

### 3. Configure Gemini API Key

Create a `.env` file in the project root directory and add:

```env
GEMINI_API_KEY=your_gemini_api_key_here
```

### 4. Run the Application

```bash
python app.py
```

### 5. Open in Browser

```
http://127.0.0.1:5000
```
